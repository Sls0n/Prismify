## Prismify

Prismify is a web app that aims to revitalize & enhance boring images/screenshots. With prismify, you can effortlessly enhance your images/screenshots.

## Tech Stacks

- Typescript
- Next
- React
- Tailwind
- Prisma
- MySQL
- PlanetScale
